# 超市管理系统 java94

## 项目概述

本项目为超市管理系统，基于前后端分离的架构模式，采用Java和SpringBoot技术栈进行后端开发，前端采用Vue框架。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
  - navicat 17
  - node 14
  - Redis
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Vue
  - Redis
